# Visão de Produto

## Nome provisório

Python AI Assistant

## Proposta

Uma aplicação de chat com IA focada em auxiliar desenvolvedores Python. O usuário poderá conversar com o assistente, enviar arquivos, imagens ou áudio, selecionar o nível de raciocínio desejado e consultar respostas com exemplos de código.

## Personas

### Desenvolvedor iniciante em Python

Quer explicações simples, exemplos práticos e correções de código.

### Desenvolvedor backend

Quer ajuda com Flask, FastAPI, SQLAlchemy, testes, arquitetura e boas práticas.

### Avaliador técnico

Quer verificar se a prova foi implementada com qualidade, documentação, testes e arquitetura.

## Funcionalidades principais

| Funcionalidade | Descrição |
|---|---|
| Chat textual | Usuário digita perguntas sobre Python |
| Modo thinking | Escolha entre rápido, equilibrado e profundo |
| Upload de documentos | Envio de PDF, TXT, Markdown e arquivos de código |
| Upload de imagens | Envio de prints de erro, diagramas ou código |
| Áudio | Gravação ou upload de áudio para pergunta |
| Histórico | Lista de conversas na sidebar |
| Tema claro/escuro | Alternância visual persistente |
| Layout responsivo | Desktop, tablet e mobile |
| Streaming | Resposta progressiva do assistente |
| Biblioteca virtual | API de livros exigida pela prova |
| Busca semântica | Recuperação de documentos por similaridade |

## Não objetivos do MVP

- Clonar marca, nome ou identidade visual de terceiros.
- Autenticação complexa.
- Multi-tenant avançado.
- Marketplace de plugins.
- Billing.
- Mobile app nativo.

## Resultado esperado

Ao final, o projeto deverá demonstrar:

- domínio backend;
- integração com IA generativa;
- integração com embeddings e vector store;
- frontend moderno e testável;
- cuidado com UX;
- documentação profissional.
